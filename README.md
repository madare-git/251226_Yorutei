# 📘 README: よる定（YORUTEI）— 夜の定食リアルタイムプラットフォーム

## 🌃 1. 概要 (What is YORUTEI?)
「よる定（YORUTEI）」は、「夜にアルコールなしで定食を食べたい利用者」と「アイドルタイムの売上を増やし、食材ロスを減らしたい飲食店」をリアルタイムでマッチングするWeb/モバイルプラットフォームです。

## ✨ 2. 提供価値 (Value Proposition)
### ユーザー（利用者）への価値
- **瞬時な発見**: 「今、夜定食を提供している店」だけが地図上に表示されます。
- **安心価格**: アルコール注文不要、お通しなし（¥1,000〜¥1,500）。
- **高品質**: 店の料理力を活かした「おまかせ定食」が楽しめます。

### 店舗（ベンダー）への価値
- **収益改善**: 夜のアイドルタイムの売上向上。
- **フードロス削減**: 在庫食材をおまかせ定食に転用可能。
- **簡単オペレーション**: ON / OFF ワンタップで提供開始可能。

## 📍 3. アプリケーション機能概要

### A. ユーザー向けMAP機能
今後実装（登録された店舗データを元にリアルタイムマッピングを予定）

### B. 店舗向け管理画面機能
店舗側の運用負担を最小限に抑えつつ、正確な情報をユーザーに届けるための基本機能を備えています。

#### 1. 店舗基本情報の新規登録 (`index.php`)
店舗の「看板」となる基本情報を一括で登録します。
* **登録項目**: 代表者氏名、店舗名、住所、電話番号、メールアドレス、主要ジャンルを登録可能です。
* **ポリシー設定**: お子様の入店ポリシー（制限なし・年齢制限・NG）やアレルギー対応の可否（可・要相談・不可）など、ユーザーが店選びで重視する情報を選択式で設定できます。
* **入力サポート**: メールアドレス入力欄には `type="email"` を採用。ブラウザによる形式チェックと必須入力（`required`）を完備し、誤入力を未然に防ぎます。

#### 2. 重複登録防止・データセキュリティ (`write.php`)
信頼性の高いデータベース運用のためのバックエンド処理を行います。
* **リアルタイム書き込み**: 入力データは日時スタンプと共に `data/data.txt` へ即座に蓄積されます。
* **重複チェック機能**: 登録実行時に、既に登録済みのメールアドレスがファイル内に存在するかを自動スキャンします。
* **エラーハンドリング**: 重複が検知された場合は、JavaScriptによる警告ポップアップを表示し、ブラウザバックで修正を促すことでデータの整合性を保ちます。

#### 3. 登録情報の照会・確認 (`read.php`)
登録した情報をいつでも確認できるマイページ機能です。
* **メールアドレス検索**: セキュリティと利便性を考慮し、登録済みのメールアドレスを入力することで該当店舗の情報のみを抽出して表示します。
* **モバイル最適化レイアウト**: 照会結果は、現場のスタッフがスマートフォンからでも一目で確認しやすい「縦型テーブル形式」を採用しています。
* **共通UIシステム**: 外部CSS（`style.css`）を適用し、登録から確認まで一貫したデザイン体験を提供します。

## 🚀 4. 今後の拡張予定（ロードマップ）
* **ステータス切替機能**: 「定食提供中 / 終了」をワンタップで切り替え、MAPへ即時反映。
* **情報の編集・更新機能**: 照会画面（`read.php`）から直接登録内容を修正・上書きできるエディタ機能。
* **写真アップロード**: 本日の定食イメージや店舗外観を公開する画像投稿機能。
* **認証機能**: メールアドレスだけでなく、パスワードによる安全なログイン管理。
